# Telegram Salary Bot

Google Drive’dan maosh fayllarini yuboradigan Telegram bot.

## Xususiyatlari
- Yil/oy menyusi
- Google Drive integratsiyasi
- 1 soatda o‘chib ketadigan fayllar
- Forward/save cheklovlari

## Ishga tushirish (Render)
1. GitHub’ga push qiling
2. Render’da deploy qiling
3. Variables qo‘shing:
   - BOT_TOKEN
   - FOLDER_ID
   - GOOGLE_CREDS (credentials.json faylining JSON matni)
